    <div class="docs markdown"><div class="pilwrap" id="proyecto-backend-api-de-gestion-de-tareas-colaborativa">
  <h1>
    <a href="#proyecto-backend-api-de-gestion-de-tareas-colaborativa" name="proyecto-backend-api-de-gestion-de-tareas-colaborativa" class="pilcrow"></a>
🧩 Proyecto Backend — API de Gestión de Tareas Colaborativa
  </h1>
</div>
<p>API REST desarrollada con <strong>Node.js + Express + Prisma + PostgreSQL</strong>, que permite gestionar usuarios, proyectos y tareas con autenticación JWT y control de roles.</p>
<hr>
<div class="pilwrap" id="objetivo">
  <h2>
    <a href="#objetivo" name="objetivo" class="pilcrow"></a>
🚀 Objetivo
  </h2>
</div>
<p>Construir una API profesional y modular aplicando buenas prácticas de backend:</p>
<ul>
<li>Arquitectura escalable</li>
<li>Seguridad (JWT, roles)</li>
<li>Validación de datos</li>
<li>Despliegue en la nube</li>
<li>Testing básico</li>
</ul>
